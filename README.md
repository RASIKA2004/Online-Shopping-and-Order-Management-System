# 🛒 Online Shopping and Order Management System (Java)

![Language](https://img.shields.io/badge/Language-Java-red)
![Concept](https://img.shields.io/badge/Concept-OOP-blue)
![Collections](https://img.shields.io/badge/Collection-ArrayList-green)
![Application](https://img.shields.io/badge/Application-Console-brightgreen)

A console-based **Online Shopping and Order Management System** developed using **Java** and Object-Oriented Programming (OOP) concepts.

The project simulates an E-Commerce platform where users can register, browse products, manage shopping carts, place orders, process payments, generate invoices, and maintain product records.

---

## 📌 Project Overview

This project demonstrates the implementation of Java programming concepts including:

- Object-Oriented Programming
- Inheritance
- Interfaces
- Runtime Polymorphism
- Exception Handling
- ArrayList Collection Framework
- File Handling
- Multithreading
- Enums
- Utility Classes
